Coldfire Debugger Overview

The Coldfire Debugger for C/C++ is an Eclipse plugin that provides tools and interfaces to code and debug C/C++ applications. The main idea to create such a plugin is to simplify usage by interfaces of m68k tools such as gdb, gcc, etc. The binary files of command line compiler and debugger are proveided.
This plugin is based on CDT (C/C++ Development Toolkit) v4.0. For more information about Eclipse CDT, refer to www.eclipse.org/cdt.
